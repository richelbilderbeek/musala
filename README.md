# musala

Code for Musala

## Connections

From                  |To
----------------------|---------------------
Qwiic MP3 trigger 1   |Arduino pin 3
Qwiic MP3 triggers GND|Arduino GND
Arduino 3.3 V         |PIR Vin
Arduino GND           |PIR GND
Arduino 12            |PIR out
